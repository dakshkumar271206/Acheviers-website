# 🎓 Achievers Institute Web Platform

A full-stack, responsive educational web application built for the **Achievers Institute**, an academic coaching center based in Najafgarh. This platform serves a dual purpose: acting as a modern, promotional landing page to attract prospective students and functioning as a secure internal lead management system for administrative staff.

## 🚀 Key Features

### 🌐 Public-Facing Interface
* **Modern UI/UX:** Built with a "glassmorphism" aesthetic using Tailwind CSS, featuring smooth scroll behaviors, hover animations, and a responsive navigation bar.
* **Curriculum Showcase:** Detailed breakdown of academic programs (Classes 1–12, including Science and Commerce streams).
* **Facility Highlights:** Dedicated sections for the Elite Library and Faculty introductions, complete with student testimonials.

### 📈 Automated Lead Generation
* **Smart "Quick Enquiry" Modal:** A time-triggered popup form (appears after 15 seconds) designed to capture prospective student details (Name, Father's Name, Class/Course, and Contact No.).
* **Session Management:** Uses browser `sessionStorage` to ensure the lead popup doesn't repeatedly interrupt users once they've submitted their inquiry.

### 🔐 Role-Based Access & Administration
* **Secure Authentication:** Features a dedicated login portal supporting three distinct roles: `admin`, `teacher`, and `student`.
* **Admin Dashboard:** A protected route (`/admin-view`) exclusively accessible to administrators, providing a clean, tabular view of all collected student leads sorted by submission date.

## 🛠️ Tech Stack

* **Backend:** Python 3.x, Flask (v3.0.2)
* **Database:** SQLite via Flask-SQLAlchemy (v3.1.1)
    * Utilizes dual database binding (`achievers.db` for leads, `users.db` for authentication).
* **Frontend:** HTML5, Tailwind CSS (via CDN), Lucide Icons
* **Architecture:** Server-side rendering using Flask's `render_template_string` for rapid component deployment.

## ⚙️ How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/achievers-website.git](https://github.com/yourusername/achievers-website.git)
    cd achievers-website
    ```

2.  **Install dependencies:**
    Ensure you have Python installed, then install the required packages using the `requirement.txt` file.
    ```bash
    pip install -r requirement.txt
    ```

3.  **Run the application:**
    The SQLite databases (`achievers.db` and `users.db`) will automatically generate upon the first run, complete with default admin and user credentials.
    ```bash
    python app.py
    ```

4.  **Access the application:**
    Open your web browser and navigate to `http://127.0.0.1:5000/`.
